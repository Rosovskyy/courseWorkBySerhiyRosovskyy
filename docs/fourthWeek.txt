Протягом тижня викониувались роботи по роботы з API Google Maps, створення карти json з комунальними підприємствами Львова і робота з Flask. У резульаті було повторено і вивчено наступне: знаходження локації(довготи і широти) за назвою місця, знаходження дистанції між точками на карті, більш детаьно вивчені методи POST і GET. Виникло наступне питання: чи можливо якось зробити карту з обмеженими районами міста. Наприклад, у нас був json файл з к-тю населення країн, то в нас кожна країна була виділена іншим кольором. Чи можливо це якось зробити з районами міста(Сихівський, Франківський..)?
